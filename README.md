# 🌾 AgriMind — 作物智能会诊系统

**团队：禾智** | 西北农林科技大学

基于多智能体辩论协议（DDP）的可解释农业诊断系统，将作物诊断从图像分类问题重新定义为多模态推理问题。

## 核心创新

1. **诊断辩论协议 (DDP)** — 初诊、质疑、仲裁三个AI专家结构化辩论
2. **主动视觉问诊 (AVD)** — AI主动引导用户拍摄关键视角
3. **AgriReason 评测基准** — 农业多模态推理评测数据集

## 技术栈

- Qwen2.5-VL-7B + QLoRA 微调
- FastAPI + WebSocket
- React + TailwindCSS + Framer Motion
- ChromaDB + BGE-M3
